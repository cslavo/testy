# Testy

Zbierka testovacích HTML súborov pre rôzne účely. Projekt slúži na zhromažďovanie HTML súborov, ktoré sa publikujú priamo z repozitára.

## Obsah

- **index.html** - Hlavná stránka s navigáciou
- **test-forms.html** - Testovanie formulárových prvkov
- **test-tables.html** - Ukážky rôznych štýlov tabuliek
- **test-animations.html** - CSS animácie a hover efekty

## Použitie

Stránky je možné prehliadať priamo cez GitHub Pages alebo lokálne otvorením `index.html` v prehliadači.
